# Раздатчик почт для Telegram

Бот отдаёт адреса по одному и хранит их в SQLite. После выдачи адрес больше никому не попадёт, включая случаи одновременных запросов.

## Запуск

1. Установите Python 3.10+.
2. В PowerShell из папки проекта выполните:

   ```powershell
   py -m venv .venv
   .\.venv\Scripts\Activate.ps1
   pip install -r requirements.txt
   $env:BOT_TOKEN="токен_от_BotFather"
   $env:ADMIN_IDS="ваш_числовой_telegram_id"
   $env:ACCESS_PASSWORD="придумайте-сложный-пароль"
   python bot.py
   ```

   Узнать свой числовой Telegram ID можно через бота `@userinfobot`. Для нескольких администраторов укажите ID через запятую.

3. Откройте вашего бота и выполните `/start`.

## Как загрузить адреса

Администратор отправляет боту документ `.txt`, где один адрес на строку:

```text
gegwgwegw@gmail.com
grwehgewrherh@gmail.com
```

Бот добавит только новые корректные адреса. Команда `/stock` показывает остаток. База находится в `emails.db`; не удаляйте её, если хотите сохранить историю выданных адресов.

## Доступ пользователей

При первом `/start` пользователь вводит пароль из `ACCESS_PASSWORD`. После успешной проверки его Telegram ID сохраняется в базе и пароль повторно не запрашивается. Чтобы запросить пароль у всех заново, остановите бота и удалите из `emails.db` таблицу `allowed_users` (это не затронет список адресов и историю выдачи).
